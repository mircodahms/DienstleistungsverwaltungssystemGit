# DienstleistungsverwaltungssystemGit

Hauptfunktionen:
Einloggen -> Dashbord
-> Dienst erstellen
-> Diensteübersicht -> Diest bearbeiten/ löschen
-> Nutzungsdaten -> Nutzungsdaten bearbeiten/ löschen & neuen Nutzer anlegen & neues spezifisches Attribut anlegen
